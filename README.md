# OpenMC Performance Benchmark Results

This repository stores the results of automated performance benchmarks for [OpenMC](https://github.com/openmc-dev/openmc), run using [Airspeed Velocity (ASV)](https://asv.readthedocs.io/).

Benchmarks are run automatically on every commit to the `develop` branch of OpenMC via the [openmc-performance-bench](https://github.com/openmc-dev/openmc-performance-bench) repository.

## Viewing Results

The benchmark dashboard is published at https://openmc-dev.github.io/openmc-performance-bench-results

## Running Benchmarks

To run the benchmarks yourself or contribute, see the [openmc-performance-bench](https://github.com/openmc-dev/openmc-performance-bench) repository.
